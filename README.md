# Python 3 AI Trader Predictor

## Installation
1. Clone repo
```
git clone https://github.com/sudowoodo-keym/trading_ai.git
```
2. Create venv
```
python3 -m venv venv
```
3. Activate venv
```
source /venvbin/activate
```
4. Install dependencies
```
pip install -r dependencies.txt
```
5. Run program
```
python3 quant_trading.py
```
